## Notable findings from the first round

- **Pass 2 < Pass 1 in tokens** (~0.86× Pass 1) — opposite of the 2026-06-08 code-tooling result where Pass 2 was ~1.4× Pass 1. Hypothesis: Pass 1's checklist character required more file reads (10 tool uses) than Pass 2's argument-shape character (5 tool uses). At code-tooling scale the artefacts are short and Pass 2's longer reasoning dominated; at paper-scale prose Pass 1's broader checklist sweep dominated. Worth a third data point at mixed scale to test which side of the prose/code threshold matters.
- **Zero overlap on load-bearing findings between Pass 1 and Pass 2** (paper-scale prose, 2026-06-11). First paper-scale prose replication of DR-011's disjoint-coverage prediction at intra-family scale. The two sets are structurally disjoint, not coincidentally:
  - Pass 1 caught: stale path references, format consistency, freshness markers, procedural-discipline gaps, label-vs-tier tensions (checklist-rigour failure modes)
  - Pass 2 caught: warrant-counter omissions, aspirational framing, motivated retrospection, principled-distinction edge cases, principle-overreach (argument-shape failure modes)
  - Logged in DR-011 Evidence Base 2026-06-11 as the first paper-scale prose confirmation at intra-family.
- **Pass 3 cross-vendor (Gemini) — 3-4 novel findings AND 4 overlap findings with Pass 2** (paper-scale prose, 2026-06-11). The intra-family "essentially disjoint" framing in DR-011 Context does *not* extend cleanly to cross-vendor at this scale — Pass 3 hit Pass 2 directly on the v2.1.1 CHANGELOG "over-cautious" finding and partially overlapped on three more. *But* the underlying `memory/hypothesis-log.md` position (Pass 3 finds ≥1 novel load-bearing item that neither intra-family pass found) holds with 3-4 novel items: Toulmin/Whetten jargon undefined, Step 6 Delegation Paradox, three-way structural cut alternative, terminology micro-violations Pass 1 missed. The stronger "fully disjoint" prediction is refined; the weaker "≥1 novel" prediction is confirmed. Methodological caveat: Pass 3's lower token cost (~21K vs 81.5K / 69.7K) reflects inline content delivery, not cross-vendor efficiency — Passes 1+2 navigated files via tool use, Pass 3 received them in the prompt. Logged in DR-011 Evidence Base 2026-06-11 as the first paper-scale prose data point at cross-family.
