# Jaakkola 2020

## Bibliographic Info
- **Authors:** Elina Jaakkola
- **Year:** 2020
- **Title:** Designing conceptual articles: four approaches
- **Venue:** AMS Review, 10(1-2), 18-26
- **DOI:** 10.1007/s13162-020-00161-0

## Summary
Identifies four approaches to writing conceptual/theoretical articles: theory synthesis (integrating across literature streams), theory adaptation (amending existing theory), typology (classifying conceptual variants), and model (building a framework predicting relationships). Each approach has different building blocks and different verification needs.

## Key Findings
- Theory Synthesis: contribution is the integration across multiple theories/streams
- Theory Adaptation: contribution is the amendment using insights from other theories
- Typology: contribution is the classification scheme itself
- Model: contribution is the framework predicting relationships between concepts
- Each approach has a different "what's new" -- and each would need different verification

## Relevance to DR-004
Shows that even within "conceptual papers" there are at least four distinct contribution types, each with different atomic units. A one-size-fits-all registry extension won't work. The registry model needs to be flexible enough to accommodate synthesis claims, adaptation rationale, classification criteria, and model propositions.
